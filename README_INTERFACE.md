# Interface locale Dealer Quote Manager

## Installation

Depuis le dossier du projet :

```powershell
pip install -r requirements.txt
```

## Lancement

```powershell
cd backend/app
python main.py
```

Puis ouvrir dans le navigateur :

```text
http://127.0.0.1:8000
```

## Fonctions

- Historique des devis
- Import fichier ServiceCalculationExport.xlsx
- Calcul pricing dealer
- Export HTML
- Export PDF
- Modification paramètres dealer
